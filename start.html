提示： 建议使用本地服务器或部署到 GitHub Pages 以获得最佳体验。

方式 1: 使用 Live Server (推荐) ⭐

1. 在 VSCode 中，点击左侧扩展图标 (或按 Ctrl+Shift+X)
2. 搜索 "Live Server" (作者是 Ritwick Dey)
3. 点击 "安装"
4. 安装完成后，右键任意 HTML 文件 → "Open with Live Server"
5. 会自动在浏览器中打开应用，并支持热更新
✅ 这是最简单的方式！ 不需要安装 Python 或 Node.js，一键启动服务器。

方式 2: 使用 Python 服务器

1. 打开命令提示符或 PowerShell
2. 输入：cd e:\project_20260401_020123\projects
3. 输入：python -m http.server 8000
4. 在浏览器访问：http://localhost:8000

方式 3: 部署到 GitHub Pages (永久在线) 🌐

1. 访问 GitHub 并登录
2. 创建新仓库 (如 ai-chat-app)
3. 上传所有项目文件
4. Settings → Pages → Branch 选择 main → Save
5. 等待 2 分钟，生成 https://你的用户名.github.io/ai-chat-app/
6. 可以分享给任何人使用！
